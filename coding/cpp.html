C++ Temel Eğitimi ve Mülakat Soruları

C++'ın Temelleri

C++ Nedir?

C++, C dilinin üzerine inşa edilmiş, nesne yönelimli, genel amaçlı bir programlama dilidir. C++ dilinin temel özellikleri şunlardır:

  • Nesne Yönelimli Programlama
  • Genel Amaçlı Kullanım
  • Düşük Seviye Bellek Yönetimi
  • Geniş Standart Kütüphane

C++ ile C Arasındaki Farklar

C, yapılandırılmış bir programlama diliyken, C++ nesne yönelimli programlamayı destekler. Ayrıca, C++ dilinde class, constructor ve destructor gibi özellikler vardır.

Veri Tipleri ve Değişkenler

Temel Veri Tipleri

  • int: Tam sayı veri tipi.
  • float: Ondalık sayı veri tipi.
  • double: Daha yüksek hassasiyetli ondalık sayı veri tipi.
  • char: Tek karakter veri tipi.
  • bool: Mantıksal veri tipi (true veya false).

Değişken Tanımlama ve Başlatma

int sayi = 10;
float oran = 3.14f;
char karakter = 'A';

Yukarıdaki örneklerde, farklı veri tiplerinde değişkenler tanımlanmış ve başlatılmıştır.

Sabitler (Constants)

const int PI = 3.14159;

Yukarıdaki örnekte, PI değişkeni sabit olarak tanımlanmıştır ve değeri değiştirilemez.

Operatörler

Aritmetik Operatörler

  • +: Toplama
  • -: Çıkarma
  • *: Çarpma
  • /: Bölme
  • %: Mod alma

Karşılaştırma Operatörleri

  • ==: Eşit mi?
  • !=: Eşit değil mi?
  • >: Büyük mü?
  • <: Küçük mü?
  • >=: Büyük eşit mi?
  • <=: Küçük eşit mi?

Mantıksal Operatörler

  • &&: Ve (And)
  • ||: Veya (Or)
  • !: Değil (Not)

Bitwise Operatörler

  • &: Ve (And)
  • |: Veya (Or)
  • ^: Özel veya (Xor)
  • ~: Ters (Not)
  • <<: Sol kaydırma (Left Shift)
  • >>: Sağ kaydırma (Right Shift)
Kontrol Yapıları

If-Else

int sayi = 10;
if (sayi > 5) {
    std::cout << "Sayi 5'ten büyük." << std::endl;
} else {
    std::cout << "Sayi 5'ten küçük veya eşit." << std::endl;
}

Switch-Case

int gun = 3;
switch (gun) {
    case 1:
        std::cout << "Pazartesi" << std::endl;
        break;
    case 2:
        std::cout << "Salı" << std::endl;
        break;
    case 3:
        std::cout << "Çarşamba" << std::endl;
        break;
    default:
        std::cout << "Geçersiz gün" << std::endl;
}

Döngüler (For, While, Do-While)

for (int i = 0; i < 5; i++) {
    std::cout << "Merhaba" << std::endl;
}

Yukarıdaki for döngüsü, ekrana beş kez "Merhaba" yazar.

Fonksiyonlar

Fonksiyon Tanımlama ve Çağırma

int topla(int a, int b) {
    return a + b;
}

int main() {
    int sonuc = topla(5, 10);
    std::cout << "Sonuç: " << sonuc << std::endl;
    return 0;
}

Bu örnekte, topla adında bir fonksiyon tanımlanmış ve main fonksiyonunda çağrılmıştır.

Parametre Geçişi (Call by Value, Call by Reference)

void degistir(int &ref) {
    ref = 20;
}

int main() {
    int sayi = 10;
    degistir(sayi);
    std::cout << "Sayi: " << sayi << std::endl;
    return 0;
}

Bu örnekte, &ref parametresi ile değişken referans yoluyla geçirilmiş ve değeri fonksiyon içinde değiştirilmiştir.

Fonksiyon Overloading

int topla(int a, int b) {
    return a + b;
}

double topla(double a, double b) {
    return a + b;
}

Yukarıdaki örnekte, aynı isimde iki farklı fonksiyon tanımlanmış ancak parametre tipleri farklıdır. Bu duruma "function overloading" denir.

Diziler ve Pointerlar

Tek Boyutlu Diziler

int sayilar[5] = {1, 2, 3, 4, 5};
std::cout << "İlk eleman: " << sayilar[0] << std::endl;

Çok Boyutlu Diziler

int matris[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};
std::cout << "Matris elemanı: " << matris[1][2] << std::endl;

Pointerlar ve Pointer Aritmetiği

int sayi = 10;
int *ptr = &sayi;
std::cout << "Pointer adresi: " << ptr << std::endl;
std::cout << "Pointer değeri: " << *ptr << std::endl;

Bu örnekte, bir pointer tanımlanmış ve sayi değişkeninin adresi bu pointer'a atanmıştır.

Diziler ve Pointerlar Arasındaki İlişki

int dizi[5] = {10, 20, 30, 40, 50};
int *ptr = dizi;
std::cout << "İlk eleman: " << *ptr << std::endl;
std::cout << "İkinci eleman: " << *(ptr + 1) << std::endl;

Pointer'lar ve diziler arasında sıkı bir ilişki vardır. Bir dizi ismi, dizinin ilk elemanının adresini gösterir.

Nesne Yönelimli Programlama (OOP)

Sınıflar ve Nesneler

class Araba {
public:
    string marka;
    int hiz;
    
    void bilgi() {
        std::cout << "Marka: " << marka << ", Hız: " << hiz << std::endl;
    }
};

int main() {
    Araba araba1;
    araba1.marka = "BMW";
    araba1.hiz = 200;
    araba1.bilgi();
    return 0;
}

Bu örnekte, Araba adında bir sınıf tanımlanmış ve bu sınıftan bir nesne oluşturulmuştur.

Kapsülleme (Encapsulation)

Kapsülleme, sınıfın veri üyelerini private olarak tanımlayıp, bu verilere erişimi public yöntemlerle (get/set fonksiyonları gibi) kontrol etme prensibidir.

Kalıtım (Inheritance)

class Arac {
public:
    string renk;
};

class Araba : public Arac {
public:
    int hiz;
};

Yukarıdaki örnekte, Araba sınıfı Arac sınıfından kalıtım almıştır.

Polimorfizm (Polymorphism)

Polimorfizm, aynı işlevin farklı şekillerde gerçekleştirilmesidir. Örneğin, bir base sınıf fonksiyonu, derived sınıfta override edilerek farklı bir işlevsellik kazanabilir.

Abstraction ve Interface

Abstraction, karmaşık sistemleri daha basit ve anlaşılabilir hale getirmek için kullanılan bir tekniktir. Interface ise, bir sınıfın dışarıya nasıl görüneceğini belirleyen bir yapıdır.

Standart Şablon Kütüphanesi (STL)

Vektörler (Vectors)

#include <vector>

std::vector<int> sayilar = {1, 2, 3, 4, 5};
sayilar.push_back(6);
std::cout << "İlk eleman: " << sayilar[0] << std::endl;

Listeler (Lists)

#include <list>

std::list<int> sayilar = {1, 2, 3, 4, 5};
sayilar.push_back(6);
std::cout << "İlk eleman: " << sayilar.front() << std::endl;

Haritalar (Maps)

#include <map>

std::map<string, int> telefonRehberi;
telefonRehberi["Ahmet"] = 5551234;
std::cout << "Ahmet'in numarası: " << telefonRehberi["Ahmet"] << std::endl;
Hata Ayıklama ve İstisna Yönetimi

İstisnalar (Exceptions)

try {
    int sayi = 10;
    if (sayi == 10) {
        throw "Hata: Sayi 10 olamaz!";
    }
} catch (const char* hata) {
    std::cout << hata << std::endl;
}

Assert Makroları

#include <cassert>

int main() {
    int x = 5;
    assert(x == 5);
    return 0;
}
Giriş/Çıkış İşlemleri

Dosya İşlemleri

#include <fstream>

std::ofstream dosya("deneme.txt");
dosya << "Merhaba Dünya!";
dosya.close();

std::ifstream dosyaOku("deneme.txt");
std::string satir;
while (getline(dosyaOku, satir)) {
    std::cout << satir << std::endl;
}
dosyaOku.close();
