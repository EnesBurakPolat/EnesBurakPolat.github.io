C Kodları

Temel C Özellikleri

C, 1970'lerde Dennis Ritchie tarafından geliştirilen, genel amaçlı bir programlama dilidir. Sistem programlama, uygulama yazılımı ve gömülü sistemlerde yaygın olarak kullanılır. C'nin temel özellikleri şunlardır:

  • Fonksiyonel Programlama: C dili, fonksiyonlar kullanarak kodu yapılandırır.
  • Statik Tip Denetimi: C, derleme zamanında veri türlerini kontrol eder.
  • Pointer Kullanımı: Bellek adreslerine doğrudan erişim sağlar.
  • Performans: Düşük seviyeli işlemler ve yüksek performans sağlar.
  • Kütüphaneler: Standart kütüphanelerle birçok yerleşik fonksiyon sunar.

Örnek C Kodları

İşte C dilinde yazılmış bazı temel kod örnekleri:

Merhaba Dünya Programı

#include <stdio.h> int main() { printf("Merhaba Dünya!"); return 0; }

Faktöriyel Hesaplama

#include <stdio.h> int factorial(int n) { if (n == 0) return 1; else return n * factorial(n - 1); } int main() { int num; printf("Bir sayı girin: "); scanf("%d", &num); printf("Faktöriyel: %d\n", factorial(num)); return 0; }

Mülakat Soruları ve Cevaplar

Soru 1: C dilinin temel veri türleri nelerdir?

Cevap: C dilinde temel veri türleri şunlardır: int, float, double, char. Bunlar, tam sayılar, kesirli sayılar, büyük kesirli sayılar ve karakterleri temsil eder.

Soru 2: Pointer nedir ve nasıl kullanılır?

Cevap: Pointer, bellek adreslerini saklayan bir veri türüdür. Pointer kullanarak bellekteki adreslere erişebilir ve bu adreslerdeki verileri değiştirebiliriz. Örneğin, int *ptr bir tam sayı pointer'ıdır.

Soru 3: C dilinde bellek yönetimi nasıl yapılır?

Cevap: C dilinde bellek yönetimi, malloc, calloc, realloc ve free gibi fonksiyonlarla yapılır. malloc ve calloc dinamik bellek tahsis ederken, free bu belleği serbest bırakır.

Soru 4: C dilinde fonksiyonlar nasıl tanımlanır ve çağrılır?

Cevap: C dilinde fonksiyonlar return_type function_name(parameters) biçiminde tanımlanır. Fonksiyon çağrısı ise function_name(arguments) şeklinde yapılır. Örneğin:

int add(int a, int b) { return a + b; } int main() { int result = add(5, 3); printf("Sonuç: %d\n", result); return 0; }
