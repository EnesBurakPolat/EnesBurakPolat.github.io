CSS Kodları

Temel CSS Özellikleri

CSS (Cascading Style Sheets), web sayfalarının stilini ve düzenini tanımlamak için kullanılan bir dildir. CSS ile renkler, yazı tipleri, boşluklar ve düzenleme gibi birçok stil özelliği ayarlanabilir.

Örnek CSS Kodları

/* CSS kodu örneği */ body { font-family: Arial, sans-serif; } h1 { color: blue; }

Özellikler ve Kullanım

CSS ile birçok stil özelliği uygulanabilir. İşte bazı temel özellikler:

  • color: Metin rengini belirler.
  • background-color: Arka plan rengini belirler.
  • font-family: Yazı tipini belirler.
  • margin: Elemanlar arasında boşluk oluşturur.
  • padding: Elemanın içindeki boşluğu belirler.
  • border: Elemanlara sınır ekler.

Mülakat Soruları ve Cevaplar

Soru 1: CSS nedir ve neden kullanılır?

Cevap: CSS (Cascading Style Sheets), web sayfalarının stilini tanımlamak için kullanılan bir dildir. CSS, HTML içeriğine biçim ve düzen ekler, bu da web sayfalarının daha çekici ve okunabilir olmasını sağlar.

Soru 2: CSS seçicileri nedir ve nasıl kullanılır?

Cevap: CSS seçicileri, sayfadaki HTML öğelerini seçmek için kullanılır. Örneğin, p seçicisi tüm paragraf öğelerini, #id seçicisi belirli bir id'ye sahip öğeleri, .class seçicisi belirli bir sınıfa sahip öğeleri seçer.

Soru 3: CSS ile bir elemanın yalnızca üst ve alt kenarlarına boşluk eklemek için hangi özellik kullanılır?

Cevap: margin-top ve margin-bottom özellikleri, bir elemanın yalnızca üst ve alt kenarlarına boşluk eklemek için kullanılır.

Soru 4: Flexbox nedir ve ne zaman kullanılır?

Cevap: Flexbox, CSS ile esnek ve dinamik düzenler oluşturmak için kullanılan bir düzenleme modelidir. İçerik alanları arasında boşluk dağıtmak, hizalamak ve düzenlemek için kullanılır. Özellikle responsive tasarımlar için uygundur.
