Flask Temel Eğitim

Flask'e Giriş

Flask, Python ile yazılmış hafif ve esnek bir web framework'üdür. Web uygulamaları geliştirmek için kullanılır.

# Flask kurulum komutu
pip install flask
Basit Bir Flask Uygulaması

Flask ile basit bir "Merhaba Dünya!" uygulaması oluşturmak için aşağıdaki kodu kullanabilirsiniz:

from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Merhaba Dünya!"

if __name__ == '__main__':
    app.run(debug=True)
URL Yönlendirmeleri

Flask'te farklı URL yollarına yönlendirme yapmak için `@app.route` dekoratörü kullanılır.

@app.route('/hakkinda')
def hakkinda():
    return "Hakkında Sayfası"
HTML Şablonları

Flask, Jinja2 şablon motorunu kullanarak HTML şablonları ile dinamik içerik oluşturmayı destekler.

from flask import render_template

@app.route('/merhaba')
def merhaba():
    return render_template('merhaba.html', isim='Ali')

Örnek HTML şablonu (merhaba.html):

<html>
    <body>
        Merhaba, {{ isim }}!
    </body>
</html>
Formlar

Flask ile form verilerini almak için `request` objesi kullanılır.

from flask import request

@app.route('/form', methods=['GET', 'POST'])
def form():
    if request.method == 'POST':
        veri = request.form['input']
        return f"Gelen veri: {veri}"
    return '''
        <form method="post">
            <input type="text" name="input">
            <input type="submit">
        </form>
    '''
Flask Mülakat Soruları

Soru 1: Flask nedir?

Flask, Python programlama dili ile yazılmış hafif bir web framework'üdür.

Soru 2: Flask ile nasıl bir web uygulaması başlatılır?

Flask ile bir uygulama başlatmak için `Flask` sınıfından bir örnek oluşturulur ve `app.run()` metodu çağrılır.

Soru 3: `@app.route` dekoratörü ne işe yarar?

`@app.route` dekoratörü, bir URL yolunu belirli bir fonksiyon ile ilişkilendirir ve bu URL'ye gelen isteklerde ilgili fonksiyon çalıştırılır.

Soru 4: Flask'te nasıl form verisi alınır?

Form verileri `request.form` objesi aracılığıyla alınır.

Soru 5: Jinja2 şablon motoru nedir?

Jinja2, Flask tarafından kullanılan bir şablon motorudur ve HTML şablonları ile dinamik içerik oluşturmayı sağlar.
