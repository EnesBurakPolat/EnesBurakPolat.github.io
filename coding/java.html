Java Temel Eğitim

Java'ya Giriş

Java, 1995 yılında Sun Microsystems tarafından geliştirilen, nesne yönelimli bir programlama dilidir. Platform bağımsızdır ve "Write Once, Run Anywhere" (Bir kere yaz, her yerde çalıştır) mottosuyla bilinir.

public class Main {
    public static void main(String[] args) {
        System.out.println("Merhaba Dünya!");
    }
}
Değişkenler ve Veri Tipleri

Java'da veri tipleri temel (primitive) ve referans tipler olarak ikiye ayrılır. Temel veri tipleri arasında int, double, char gibi tipler yer alır. Referans tipler ise String ve nesneler gibi daha kompleks veri yapılarıdır.

int sayi = 10;
double ondalik = 5.5;
String metin = "Merhaba";
Karar Yapıları

Karar yapıları, programın belirli koşullara göre farklı yollar izlemesini sağlar. Java'da if-else ve switch-case gibi yapılandırmalar bu amaçla kullanılır.

If-Else

int sayi = 5;
if (sayi > 5) {
    System.out.println("Sayi 5'ten büyük.");
} else {
    System.out.println("Sayi 5'ten küçük veya eşit.");
}

Switch-Case

int gun = 3;
switch (gun) {
    case 1:
        System.out.println("Pazartesi");
        break;
    case 2:
        System.out.println("Salı");
        break;
    case 3:
        System.out.println("Çarşamba");
        break;
    default:
        System.out.println("Geçersiz gün");
}
Döngüler

Döngüler, belirli bir koşul sağlandığı sürece aynı kod bloğunu tekrar tekrar çalıştırır. Java'da for, while ve do-while gibi döngüler bulunur.

For Döngüsü

for (int i = 0; i < 5; i++) {
    System.out.println("Merhaba");
}

While Döngüsü

int i = 0;
while (i < 5) {
    System.out.println("Merhaba");
    i++;
}

Do-While Döngüsü

int i = 0;
do {
    System.out.println("Merhaba");
    i++;
} while (i < 5);
Fonksiyonlar

Fonksiyonlar, bir işlemi gerçekleştirmek üzere tanımlanmış kod bloklarıdır. Java'da fonksiyonlar metodlar olarak adlandırılır ve bir sınıf içerisinde tanımlanır.

Fonksiyon Tanımlama ve Çağırma

public static int topla(int a, int b) {
    return a + b;
}

public static void main(String[] args) {
    int sonuc = topla(5, 10);
    System.out.println("Sonuç: " + sonuc);
}
Diziler

Diziler, aynı tipteki birden fazla veriyi saklamak için kullanılır. Java'da diziler sabit boyutludur ve bir kez tanımlandıktan sonra boyutu değiştirilemez.

Tek Boyutlu Diziler

int[] sayilar = {1, 2, 3, 4, 5};
System.out.println("İlk eleman: " + sayilar[0]);

Çok Boyutlu Diziler

int[][] matris = {
    {1, 2, 3},
    {4, 5, 6}
};
System.out.println("Matris elemanı: " + matris[1][2]);
Nesne Yönelimli Programlama (OOP)

OOP, yazılım geliştirmede verilerin ve fonksiyonların birlikte ele alındığı bir yaklaşımdır. Java, OOP prensiplerini tamamen destekler ve sınıflar, nesneler, kalıtım, çok biçimlilik gibi kavramlar içerir.

Sınıflar ve Nesneler

class Araba {
    String marka;
    int hiz;
    
    void bilgi() {
        System.out.println("Marka: " + marka + ", Hız: " + hiz);
    }
}

public class Main {
    public static void main(String[] args) {
        Araba araba1 = new Araba();
        araba1.marka = "BMW";
        araba1.hiz = 200;
        araba1.bilgi();
    }
}
Hata Ayıklama ve İstisna Yönetimi

Hata ayıklama, yazılım geliştirme sürecinde kodun doğru çalışıp çalışmadığını kontrol etmek için yapılan işlemdir. İstisna yönetimi, programda beklenmedik hataların üstesinden gelmek için kullanılır.

İstisnalar (Exceptions)

try {
    int sayi = 10;
    if (sayi == 10) {
        throw new Exception("Hata: Sayi 10 olamaz!");
    }
} catch (Exception e) {
    System.out.println(e.getMessage());
}
Dosya İşlemleri

Java'da dosya işlemleri, verilerin dosyalara yazılması veya dosyalardan okunması gibi işlemleri içerir. `File`, `FileReader`, `FileWriter` gibi sınıflar bu işlemleri gerçekleştirmek için kullanılır.

import java.io.FileWriter;
import java.io.FileReader;

public class DosyaIslemleri {
    public static void main(String[] args) {
        try {
            // Dosya oluşturma ve yazma
            FileWriter writer = new FileWriter("dosya.txt");
            writer.write("Merhaba Dosya!");
            writer.close();

            // Dosya okuma
            FileReader reader = new FileReader("dosya.txt");
            int karakter;
            while ((karakter = reader.read()) != -1) {
                System.out.print((char) karakter);
            }
            reader.close();
        } catch (Exception e) {
            System.out.println("Bir hata oluştu: " + e.getMessage());
        }
    }
}
Java Mülakat Soruları

Soru 1: Java nedir?

Java, nesne yönelimli bir programlama dilidir ve platform bağımsızdır.

Soru 2: `final` anahtar kelimesi nedir?

`final` anahtar kelimesi, bir değişkenin değerinin değiştirilemez olduğunu, bir metodun override edilemeyeceğini ve bir sınıfın miras alınamayacağını belirtir.

Soru 3: Exception handling (istisna yönetimi) nedir?

Exception handling, programın çalışma sırasında oluşabilecek hataları ele almak ve programın beklenmedik şekilde çökmesini önlemek için kullanılan bir tekniktir.

Soru 4: `static` anahtar kelimesi nedir?

`static` anahtar kelimesi, bir değişken veya metodun sınıf düzeyinde olduğunu ve nesne oluşturmadan erişilebileceğini belirtir.

Soru 5: Java'nın platform bağımsız olmasını sağlayan nedir?

Java'nın platform bağımsız olmasını sağlayan şey, Java Sanal Makinesi (JVM) üzerinde çalışmasıdır. JVM, Java kodunu platformdan bağımsız olarak çalıştırır.
