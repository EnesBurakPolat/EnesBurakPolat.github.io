React Temel Eğitim

React'a Giriş

React, kullanıcı arayüzleri oluşturmak için kullanılan açık kaynaklı bir JavaScript kütüphanesidir. Facebook tarafından geliştirilmiştir ve bileşen bazlı bir yapı sunar.

import React from 'react';

function App() {
    return (
        <div>
            <h1>Merhaba Dünya!</h1>
        </div>
    );
}

export default App;
Bileşenler

React'te bileşenler, UI parçalarını tanımlar. Bileşenler işlevsel (fonksiyonel) ya da sınıf tabanlı olabilir.

Fonksiyonel Bileşen

function Merhaba() {
    return <h2>Merhaba!</h2>;
}

Sınıf Bileşeni

class Merhaba extends React.Component {
    render() {
        return <h2>Merhaba!</h2>;
    }
}
Props ve State

Props, bileşenlere veri geçişi yapmak için kullanılırken, state bileşenlerin kendi iç durumunu yönetir.

Props Örneği

function Kullanici(props) {
    return <p>Merhaba, {props.isim}!</p>;
}

// Kullanım
<Kullanici isim="Ali" />

State Örneği

class Sayac extends React.Component {
    constructor(props) {
        super(props);
        this.state = { sayi: 0 };
    }

    artir = () => {
        this.setState({ sayi: this.state.sayi + 1 });
    };

    render() {
        return (
            <div>
                <p>Sayı: {this.state.sayi}</p>
                <button onClick={this.artir}>Arttır</button>
            </div>
        );
    }
}
Etkinlikler

React'te, kullanıcı etkileşimleri yönetmek için etkinlik işleyicileri kullanılır. `onClick`, `onChange` gibi olaylar bu işlev için kullanılır.

function EtkinlikOrnek() {
    const [mesaj, setMesaj] = React.useState('');

    const handleClick = () => {
        setMesaj('Butona tıklandı!');
    };

    return (
        <div>
            <button onClick={handleClick}>Tıkla</button>
            <p>{mesaj}</p>
        </div>
    );
}
Formlar

Formlar, kullanıcıdan veri toplamak için kullanılır. React'te form işleme ve veri yönetimi `onChange` gibi olaylarla yapılır.

function FormOrnek() {
    const [girdi, setGirdi] = React.useState('');

    const handleChange = (event) => {
        setGirdi(event.target.value);
    };

    return (
        <div>
            <input type="text" value={girdi} onChange={handleChange} />
            <p>Girdi: {girdi}</p>
        </div>
    );
}
Listeleme

React'te listeleme, dinamik içerik görüntülemek için `map` fonksiyonu kullanılarak yapılır.

function ListeOrnek() {
    const meyveler = ['Elma', 'Armut', 'Muz'];

    return (
        <ul>
            {meyveler.map((meyve, index) => (
                <li key={index}>{meyve}</li>
            ))}
        </ul>
    );
}
React Mülakat Soruları

Soru 1: React nedir?

React, kullanıcı arayüzleri oluşturmak için kullanılan bir JavaScript kütüphanesidir. Bileşen bazlı ve hızlı bir şekilde dinamik arayüzler geliştirmeyi sağlar.

Soru 2: `props` nedir?

`props`, bileşenlere veri geçişi yapmak için kullanılan bir mekanizmadır. Bir bileşene dışarıdan veri gönderilir.

Soru 3: `state` nedir?

`state`, bileşenlerin kendi iç durumunu yönetir ve bu durum değiştiğinde bileşenin yeniden render edilmesini sağlar.

Soru 4: React'te etkinlikler nasıl işlenir?

React'te etkinlikler, `onClick`, `onChange` gibi olay işleyiciler kullanılarak işlenir. Bu işleyiciler, kullanıcı etkileşimlerine yanıt verir.

Soru 5: React'te form verileri nasıl işlenir?

React'te form verileri, `onChange` gibi olaylarla işlenir ve bu veriler `state` kullanılarak yönetilir.
