JavaScript Temel Eğitim

JavaScript'e Giriş

JavaScript, web sayfalarına etkileşim eklemek için kullanılan bir programlama dilidir. Tarayıcıda çalışır ve dinamik içeriği yönetmek için kullanılır.

console.log("Merhaba Dünya!");
Değişkenler ve Veri Tipleri

JavaScript'te değişkenler `var`, `let`, ve `const` anahtar kelimeleri ile tanımlanabilir. Veri tipleri arasında string, number, boolean ve object bulunur.

let isim = "Ali";
const yas = 30;
let dogruMu = true;
let kisi = { ad: "Ali", yas: 30 };
Karar Yapıları

JavaScript'te karar yapıları, belirli koşullara göre farklı kod bloklarının çalışmasını sağlar.

if (yas > 18) {
    console.log("Yetişkin");
} else {
    console.log("Çocuk");
}
Döngüler

Döngüler, bir kod bloğunu belirli bir sayıda veya belirli bir koşul sağlanana kadar tekrar eder.

for Döngüsü

for (let i = 0; i < 5; i++) {
    console.log(i);
}

while Döngüsü

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}
Fonksiyonlar

JavaScript'te fonksiyonlar, belirli bir işlevi yerine getiren kod bloklarıdır. `function` anahtar kelimesi ile tanımlanır.

function selamla(isim) {
    return "Merhaba, " + isim + "!";
}

console.log(selamla("Ali"));
Diziler

Diziler, birden fazla değeri tek bir değişkende saklamayı sağlar. JavaScript'te diziler sıfırdan başlayan indekslerle erişilir.

let meyveler = ["Elma", "Armut", "Muz"];
console.log(meyveler[0]); // Elma
Nesne Yönelimli Programlama (OOP)

JavaScript, nesne yönelimli programlamayı destekler. Nesneler, özellikler (properties) ve metotlar (methods) içerir.

class Kisi {
    constructor(ad, yas) {
        this.ad = ad;
        this.yas = yas;
    }

    tanit() {
        return `Adım ${this.ad} ve yaşım ${this.yas}`;
    }
}

let kisi = new Kisi("Ali", 30);
console.log(kisi.tanit());
Hata Ayıklama ve İstisna Yönetimi

JavaScript'te hata ayıklama ve istisna yönetimi, `try`, `catch` ve `finally` blokları kullanılarak yapılır.

try {
    throw new Error("Bir hata oluştu!");
} catch (e) {
    console.error(e.message);
} finally {
    console.log("Hata ayıklama tamamlandı.");
}
JavaScript Mülakat Soruları

Soru 1: JavaScript nedir?

JavaScript, web sayfalarını dinamik hale getiren ve etkileşimli kullanıcı arayüzleri oluşturan bir programlama dilidir.

Soru 2: `var`, `let`, ve `const` arasındaki fark nedir?

`var`, ES5'te tanımlanmıştır ve fonksiyon kapsamına sahiptir. `let` ve `const` ES6'da tanımlanmış olup, blok kapsamına sahiptir. `const` değişmez değişkenler için kullanılır.

Soru 3: JavaScript'te bir fonksiyon nasıl tanımlanır?

Fonksiyonlar `function` anahtar kelimesi ile tanımlanır ve bir isim, parametre listesi ve gövde içerir.

Soru 4: JavaScript'te diziler nasıl tanımlanır ve kullanılır?

Diziler köşeli parantezler (`[]`) kullanılarak tanımlanır ve indeksleme ile erişilir.

Soru 5: JavaScript'te `this` anahtar kelimesi ne işe yarar?

`this`, bir nesnenin veya fonksiyonun çağrıldığı bağlama referans verir. O nesnenin veya fonksiyonun bağlamını belirtir.
