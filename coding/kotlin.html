Kotlin Temel ve Orta Seviye Eğitim

Kotlin'e Giriş

Kotlin, JVM (Java Virtual Machine) üzerinde çalışan modern ve açık kaynak bir programlama dilidir. Java ile tamamen uyumludur ve Android geliştirme için yaygın olarak kullanılır.

fun main() {
    println("Merhaba Kotlin!")
}
Değişkenler ve Veri Tipleri

Kotlin'de değişkenler `var` (değiştirilebilir) ve `val` (sabit) anahtar kelimeleri ile tanımlanır. Kotlin, veri tiplerini otomatik olarak çıkarır ancak istenirse açıkça belirtilebilir.

var isim = "Ali"
val yas = 30
var dogruMu = true
val kisi = mapOf("ad" to "Ali", "yas" to 30)
Karar Yapıları

Kotlin'de `if-else` ve `when` yapıları karar verme mekanizmaları olarak kullanılır.

If-Else

if (yas > 18) {
    println("Yetişkin")
} else {
    println("Çocuk")
}

When Yapısı

val sonuc = when (yas) {
    in 0..17 -> "Çocuk"
    18 -> "Yeni Yetişkin"
    else -> "Yetişkin"
}
println(sonuc)
Döngüler

Kotlin'de döngüler, bir kod bloğunu tekrar tekrar çalıştırmak için kullanılır.

For Döngüsü

for (i in 0..5) {
    println(i)
}

While Döngüsü

var i = 0
while (i < 5) {
    println(i)
    i++
}
Fonksiyonlar

Kotlin'de fonksiyonlar `fun` anahtar kelimesi ile tanımlanır ve belirli bir işlevi yerine getirir.

fun selamla(isim: String): String {
    return "Merhaba, $isim!"
}

println(selamla("Ali"))
Diziler ve Listeler

Kotlin'de diziler ve listeler, birden fazla değeri saklamak için kullanılır. Listeler, mutable (değiştirilebilir) veya immutable (değiştirilemez) olabilir.

Diziler

val meyveler = arrayOf("Elma", "Armut", "Muz")
println(meyveler[0])  // Elma

Listeler

val sehirler = listOf("Ankara", "İstanbul", "İzmir")
println(sehirler[1])  // İstanbul
Nesne Yönelimli Programlama (OOP)

Kotlin, nesne yönelimli programlamayı destekler. Sınıflar, özellikler ve metotlar tanımlayarak nesneler oluşturabilirsiniz.

class Kisi(val ad: String, val yas: Int) {
    fun tanit(): String {
        return "Adım $ad ve yaşım $yas"
    }
}

val kisi = Kisi("Ali", 30)
println(kisi.tanit())
Hata Ayıklama ve İstisna Yönetimi

Kotlin'de istisna yönetimi `try`, `catch`, ve `finally` blokları ile yapılır. Hataları kontrol etmek ve programın çökmesini engellemek için kullanılır.

try {
    throw Exception("Bir hata oluştu!")
} catch (e: Exception) {
    println(e.message)
} finally {
    println("Hata yönetimi tamamlandı.")
}
Kotlin Mülakat Soruları

Soru 1: Kotlin nedir?

Kotlin, JVM üzerinde çalışan modern bir programlama dilidir ve özellikle Android geliştirme için kullanılır.

Soru 2: `var` ve `val` arasındaki fark nedir?

`var` ile tanımlanan değişkenler değiştirilebilirken, `val` ile tanımlanan değişkenler sabittir ve ilk değer verildikten sonra değiştirilemez.

Soru 3: Kotlin'de bir fonksiyon nasıl tanımlanır?

Fonksiyonlar `fun` anahtar kelimesi ile tanımlanır. İsim, parametre listesi ve dönüş tipi içerir.

Soru 4: Kotlin'de diziler ve listeler nasıl kullanılır?

Diziler `arrayOf` ile, listeler ise `listOf` ile tanımlanır. Mutable listeler için `mutableListOf` kullanılır.

Soru 5: Kotlin'de `when` yapısı ne işe yarar?

`when` yapısı, koşul ifadelerinin daha temiz bir şekilde yazılmasını sağlar. `switch-case` yapısına benzer.
